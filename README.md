# CF Banque Investment

Site institutionnel de CF Banque Investment — institution financière internationale spécialisée en banque d'investissement, marchés de capitaux, conseil financier stratégique et gestion d'actifs institutionnelle.

## Stack technique

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui + Radix UI
- React Router v6

## Démarrage

```bash
npm install
npm run dev
```
